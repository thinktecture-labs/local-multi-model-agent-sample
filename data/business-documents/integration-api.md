---
title: Nextera API & Integrations
category: integration
---
Nextera exposes a fully OpenAI-compatible REST API. Any existing application built for OpenAI's API works with Nextera with a single base_url change — no code modifications needed. The API supports: chat completions, embeddings, function calling, streaming, and batch requests. Official SDKs: Python, TypeScript/Node, Go, and .NET C#. Native integrations: LangChain, LlamaIndex, Semantic Kernel, Haystack, and AutoGen. Webhooks are available on Professional+.
