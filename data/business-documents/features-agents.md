---
title: Agentic Pipeline Features
category: features
---
Nextera supports building multi-step AI agents with tool use. The agent framework includes: intent classification, tool registry, function calling (OpenAI-compatible), ReAct-style reasoning loops, multi-agent orchestration, and execution tracing for debugging. Built-in tools: SQL query, vector search, web search (optional), code execution (sandboxed), and HTTP API calls. Custom tools can be added in Python with a simple base class. Supports Gemma, Llama, Mistral, and Qwen model families.
