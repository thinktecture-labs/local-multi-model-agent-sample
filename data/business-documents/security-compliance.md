---
title: Security and Compliance
category: security
---
Nextera is designed for organizations with strict data security requirements. All data processing is local — no telemetry, no cloud calls, no model API usage. Compliance: SOC 2 Type II (Enterprise), GDPR-ready architecture, ISO 27001 alignment documentation, and HIPAA BAA available on Enterprise. Security features: end-to-end encryption at rest (AES-256), TLS 1.3 in transit, role-based access control (RBAC), audit logging, secrets management via Vault integration, and regular third-party penetration testing.
