## Native runtime dependencies

### llama.cpp

- Repository: <https://github.com/ggerganov/llama.cpp>
- License: **MIT** — see `vendor/llama.cpp/LICENSE`
- Integration: vendored as a Git submodule under `vendor/llama.cpp`
  ([`.gitmodules`](.gitmodules)). Built into a `llama-server` binary by
  [`scripts/build_llama.sh`](scripts/build_llama.sh). The agent talks to it
  over llama.cpp's HTTP API (OpenAI-compatible) — no llama.cpp source code
  is copied into this repo.
- Pinned commit: see `git submodule status vendor/llama.cpp`.
